# 🌊 OceanRoute Nav – Maritime Navigation Optimizer

A maritime route optimization and navigation simulation system built using C++ with SFML/GLUT visualization.

## Team Members 
1. Novwera Amjad 
2. Fahad Shoaib 

## 🚢 Features
- Graph-based maritime routing system
- Custom data structures (No STL)
- Route parsing from text files
- Shortest path algorithms (Dijkstra, A*)
- Port docking queues
- Multi-leg route booking
- Subgraph filtering
- Visual route animation
- Maritime traffic simulation

## 🧠 Core Concepts
- Graph theory
- Pathfinding algorithms
- Queue systems
- Simulation modeling
- Real-world maritime logistics
- Low-level memory management (manual structures)

## 📁 Data Files
- `Routes.txt` → Maritime routes
- `PortCharges.txt` → Port docking costs
- `Ports.txt` → Port metadata

## 🛠 Tech Stack
- Language: C++
- Graphics: SFML / GLUT
- No STL (custom structures)
- File parsing
- Manual memory management

## 🧩 Architecture
- Graph Engine
- Routing Engine
- Simulation Engine
- Visualization Engine
- Booking Engine

## 🎯 Project Vision
To simulate real-world maritime navigation, logistics optimization, and intelligent routing using algorithmic and systems-level programming.
